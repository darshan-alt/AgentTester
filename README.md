# Agent Tester

Agent Tester is a lightweight, frontend-only web application designed to evaluate the responses generated by various AI agents (LLMs). It allows users to input a prompt and an agent's answer, and then uses an LLM of their choice to evaluate the answer line-by-line for accuracy, reasoning errors, and guardrail violations.

## Features
- **Frontend Only:** Runs entirely in the browser using React and Vite. No backend is required.
- **Line-by-Line Evaluation:** Evaluates each line of an AI's answer, checking for correctness and guardrail violations (e.g., prohibited topics).
- **Multiple Providers:** Support for OpenAI, Anthropic (Claude), and Google (Gemini) as evaluating models.
- **Local Settings & API Keys:** Stores API keys and settings securely in your browser's local storage.
- **Customizable System Prompt:** Tweak the evaluator's system prompt from the settings menu to refine the evaluation output.

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.

## Deployment

This application includes a GitHub Actions workflow that automatically builds and deploys to **GitHub Pages**. Simply push to the `main` branch, and the action will handle the deployment. 

**Note:** Ensure that GitHub Pages is configured to use GitHub Actions in your repository settings:
1. Go to your repository **Settings**.
2. Navigate to **Pages** on the left sidebar.
3. Under **Build and deployment**, set the **Source** to **GitHub Actions**.
